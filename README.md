# M-M
C++ Game made by students from University of Gdansk


Gra będzie się opierać na założeniach gry Rouglelike.
Zawierać mapę stworzoną poprzez bibliotekę nethack.
Różnego rodzaju minigry pozwalające zarobić pieniądze.
Potwory, przedmioty oraz wzmocnienia.
Sklep z losowo wybranymi przedmiotami z puli do kupienia.
Mechanika turowa.
